1. Hitunglah *f(x)* untuk *x* = 1, *x* = 0, dan *x* = -2 dari fungsi$$\begin{align}
f(x) = 2x + 3 \\
\end{align}$$
	- **Penyelesaian:** $$\begin{align}
&1.\ f(1) = 2(1) + 3 = 2 + 3 = 5 \\
&2.\ f(0) = 2(0) + 3 = 0 + 3 = 5 \\
&3.\ f(-2) = 2 (-2) + 3 = -4 + 3 = -1
\end{align}$$
